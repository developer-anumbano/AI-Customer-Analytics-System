# AI Customer Analytics System

Overview
AI Customer Analytics System is a machine learning-based platform designed to analyze customer data and generate meaningful insights. The system performs data preprocessing, customer segmentation, and predictive analytics using machine learning.

This project demonstrates end-to-end AI pipeline development including data ingestion, preprocessing, model training, API development, and dashboard visualization.

Features

- Customer segmentation using machine learning
- Data preprocessing and cleaning
- CSV file upload functionality
- Backend APIs using FastAPI
- Dashboard for analytics visualization
- Customer insights generation
- End-to-end AI pipeline

Technologies Used

- Python
- FastAPI
- Machine Learning
- Pandas
- NumPy
- SQLite
- HTML
- JavaScript
